# Cluster Lab — Exercise 5

Step-by-step clustering tutorial for Movies × Users and Accounts × IPs.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- **15-step guided lesson** with live Euclidean distance math
- **Animated clustering** with Play / Pause / Step controls
- **Tap any dotted term** for definitions + relevant research papers (recommender systems, not genomics)
- **Bicluster aftermath** — why cells are excluded and what you do next
- **Explore tab** — full heatmaps for both datasets
